# ArogyaSathi

ArogyaSathi is an healthcare management platform designed to simplify medical services for patients, doctors, and administrators. The platform helps users manage appointments, medical records, prescriptions, and healthcare services in one place. ABHA ID Integration for Digital Health Identity

---

## 🚀 Features

* 🪪 ABHA ID Integration for Digital Health Identity

* 🏥 NHCX Integration for Healthcare Claim Exchange

* 🔐 Secure User Authentication

* 👨‍⚕️ Doctor & Patient Dashboard

* 📅 Appointment Booking System

* 💊 Prescription Management

* 📂 Digital Health Records

* 🔔 Notifications & Alerts

* 📱 Responsive UI for Mobile & Desktop

* ☁️ Cloud-based Data Storage

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/your-username/arogya_sathi.git
cd arogya_sathi
```

### Install Dependencies

```bash
npm install
```

or for Flutter:

```bash
flutter pub get
```

### Run the Project

#### For Web/Node.js

```bash
npm start
```

#### For Flutter

```bash
flutter run
```
